# Description

This program uses OpenGL to draw a single rectangle with smooth animation. The rectangle gradually changes its color from red to white over time. At the same time, it continuously magnifies (scales up and down) and rotates in both X and Y directions.

The animation is fully time-based, so all changes happen automatically without any user input.

# Language & Tools

C++
OpenGL (GLFW + GLAD)
VS Code
Windows OS

⚙️ How to Compile and Run

Open the terminal in the project folder.
Compile the program using the compiler setup.
Run the generated executable file.

# Output
One rectangle
Smooth color transition (red → white)
Continuous scaling (magnification)
Rotation in X and Y direction
Time-based automatic animation
